# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- TypeScript support for type safety
- Zustand store for centralized state management
- Prettier and ESLint configuration
- Comprehensive documentation (README, CONTRIBUTING, CODE_OF_CONDUCT, ARCHITECTURE)
- MIT License

### Changed
- Migrated from JavaScript to TypeScript
- Refactored state management from React hooks to Zustand
- Translated all comments and code to English
- Improved code organization following FSD architecture

### Removed
- Server-side code (project is now client-only)
- Console.log statements from production code

## [0.0.0] - Initial Release

### Added
- Basic node management functionality
- 17 different node types
- Visual graph with D3.js
- Workspace management with localStorage
- Node connections and relationships
- Material-UI components
- React Router for navigation
